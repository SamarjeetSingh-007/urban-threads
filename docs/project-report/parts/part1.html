Abstract / Project Summary

The rapid growth of e-commerce has fundamentally transformed the way consumers shop for fashion and apparel. With the increasing penetration of the internet and smartphones across India, online shopping has become a preferred mode of purchasing for the younger demographic. This project, titled "Urban Threads", addresses the growing demand for a dedicated premium streetwear e-commerce platform that caters specifically to the Indian market.

Urban Threads is a full-stack web application designed to provide users with a seamless online shopping experience for streetwear fashion. The platform allows users to browse products by category, search for specific items, view detailed product information, add items to their shopping cart and wishlist, and manage their user accounts. An integrated admin panel enables the store administrator to manage products, categories, sizes, colors, and uploaded media through a convenient web-based dashboard.

The system is built using a modern web technology stack. The frontend is developed with HTML5, CSS3, and Vanilla JavaScript, ensuring a fast and responsive user interface. The backend is powered by Node.js with Express.js, providing a robust server-side architecture. Supabase (PostgreSQL) serves as the database and file storage backend. The application is deployed on Vercel for high availability and automatic scaling.

Key features include user registration and authentication with secure password hashing (bcrypt, salt round 10), JWT-based session management, a product catalog with category-based filtering and search functionality, shopping cart with real-time updates, wishlist functionality, a contact form for user communication, responsive design for all screen sizes, and an admin dashboard for complete content management including product CRUD operations, image uploads, category management, and user oversight.

The project demonstrates the practical application of full-stack web development concepts learned during the BCA program, including requirements analysis, database design, API development, frontend engineering, authentication and authorization, testing, and cloud deployment on modern serverless platforms.

Keywords: E-commerce, Streetwear, Node.js, Express.js, Supabase, PostgreSQL, JavaScript, Full-Stack Development, REST API, JWT Authentication.
